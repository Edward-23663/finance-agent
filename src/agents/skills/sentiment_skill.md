# 舆情分析Agent技能文档

> 版本: 1.0.0
> 最后更新: 2026-03-13
> 维护者: Finance Agent Team

## 概述

舆情分析Agent负责从多渠道采集舆情数据，进行情感极性打分、热度计算、事件聚类和风险预警。

## 快速参考

| 任务类型 | 说明 | 必需参数 |
|----------|------|----------|
| `collect_sentiment` | 舆情数据抓取 | ticker |
| `sentiment_score` | 情感极性打分 | text |
| `calculate_heat` | 热度计算 | ticker |
| `event_clustering` | 事件聚类 | ticker |
| `risk_warning` | 风险预警 | ticker |

---

## 原子级功能

### 1. 舆情抓取

从向量数据库获取相关舆情数据。

**API:**
```python
task_data = {
    "type": "collect_sentiment",
    "ticker": "600519"
}
```

### 2. 情感极性打分

基于关键词匹配计算情感得分 (0-1)。

**关键词规则:**
- 正面: 增长、盈利、上涨、突破、利好、推荐、买入、增持
- 负面: 下跌、亏损、风险、利空、违规、减持、卖出、警告

### 3. 热度计算

计算舆情热度指数。

### 4. 事件聚类

将舆情事件按关键词聚类。

### 5. 风险预警规则匹配

基于负面关键词匹配触发风险预警。

---

## 组合级功能

### 舆情-股价关联分析

分析舆情与股价走势的关联性。

### 舆情长文本分段处理

对长文本进行分段向量化处理。

---

## 业务级功能

### 标的舆情风险等级评估

综合评估股票的风险等级。

**返回结果:**
```json
{
    "status": "success",
    "level": "high",
    "negative_count": 15,
    "positive_count": 5,
    "reasons": ["负面舆情较多"]
}
```

**风险等级:**
- high: 负面舆情显著多于正面
- medium: 负面舆情略多
- low: 舆情平衡或正面偏多
