# 综合思维模型Agent技能文档

> 版本: 1.0.0
> 最后更新: 2026-03-13
> 维护者: Finance Agent Team

## 概述

综合思维模型Agent融合多种思维框架进行决策分析，包括相对论分析、量子概率模型、混沌理论趋势拟合、贝叶斯概率校验等。

## 快速参考

| 任务类型 | 说明 | 必需参数 |
|----------|------|----------|
| `relative_theory` | 相对论分析因子提取 | ticker |
| `quantum_probability` | 量子概率模型 | ticker |
| `chaos_trend` | 混沌理论趋势拟合 | ticker |
| `bayesian_validation` | 贝叶斯概率校验 | ticker, prior, evidence |
| `thinking_fusion` | 多思维模型融合 | ticker |
| `nonlinear_conclusion` | 非线性结论生成 | ticker |

---

## 原子级功能

### 1. 相对论分析因子提取

提取多维度分析因子（PE、PB、ROE、增长率等），进行相对比较分析。

### 2. 量子概率模型计算

使用量子概率思维计算市场机会的确定性。

### 3. 混沌理论趋势拟合

分析价格序列的混沌特征和趋势方向。

### 4. 贝叶斯概率校验

基于先验概率和新证据计算后验概率。

---

## 组合级功能

### 多思维模型结果融合

将相对论、量子概率、混沌理论的结果进行加权融合。

**权重配置:**
- 相对论分析: 40%
- 量子概率: 30%
- 混沌理论: 30%

---

## 业务级功能

### 标的非线性分析结论生成

综合所有思维模型输出最终投资建议。

**返回结果:**
```json
{
    "status": "success",
    "ticker": "600519",
    "conclusion": "建议持有",
    "action": "hold",
    "score": 0.65,
    "confidence": 0.75,
    "risk_level": "low"
}
```

**决策规则:**
| 分数 | 置信度 | 结论 |
|------|--------|------|
| >0.7 | >0.7 | 强烈推荐买入 |
| >0.5 | >0.5 | 建议持有 |
| >0.3 | any | 建议观望 |
| <0.3 | any | 建议卖出 |
