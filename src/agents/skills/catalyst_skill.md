# 催化剂分析Agent技能文档

> 版本: 1.0.0
> 最后更新: 2026-03-13
> 维护者: Finance Agent Team

## 概述

催化剂分析Agent负责催化剂事件抓取、时间维度分类、影响力度打分、时间节点提取、催化剂-估值影响关联分析。

## 快速参考

| 任务类型 | 说明 | 必需参数 |
|----------|------|----------|
| `collect_catalyst` | 催化剂事件抓取 | ticker |
| `classify_by_time` | 时间维度分类 | ticker |
| `impact_score` | 影响力度打分 | event |
| `extract_time_node` | 时间节点提取 | event |
| `catalyst_impact_analysis` | 催化剂-估值关联 | ticker |

---

## 原子级功能

### 1. 催化剂事件抓取

从向量数据库获取催化剂事件。

### 2. 时间维度分类

将催化剂事件按时间分为短期(0-30天)、中期(30-180天)、长期(180天+)。

### 3. 影响力度打分

根据关键词判断事件影响程度。

**高影响关键词:** 收购、重组、业绩预增、订单、合作、中标
**中影响关键词:** 增持、回购、分红、扩产、投资

### 4. 时间节点提取

从文本中提取具体时间节点。

### 5. 催化剂-估值影响关联

分析催化剂事件对估值的正向/负向影响。
